Various Ethereum Token Development Standards

ERC20

Token Development

ERC 20 is a technical standard of the Ethereum token smart contract that establishes the set of guidelines for the token.

ERC223

Token Development

ERC 223 token standard is mainly used to transfer tokens to the smart contract and wallet, it prevents losses.

ERC827

Token Development

This Ethereum token standard is an enhanced version of ERC20. It supports the basic functioning of token transfer and approvals.

ERC721

Token Development

ERC721 represents token uniqueness, an advanced Non-fungible token model which cannot be duplicated and is non-interchangeable.

ERC777

Token Development

A sophisticated token standard in the Ethereum network that can empower the ERC20 via “send-token-to-contract.

ERC1155

Token Development

The very feasible token standard that you can use to create multiple NFT assets under a single smart contract.
